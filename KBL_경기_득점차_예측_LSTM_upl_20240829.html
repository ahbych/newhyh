아직 국내에서는 스포츠 경기 예측에 대한 연구가 다른 산업에 비해 활발하지 않다고 한다.
일반적 머신러닝 예측을 위한 데이터는 스포츠 경기에 있어서 미래의 경기에 해당한다.(즉 종속변수와 독립변수가 함께 발생한다.)
농구를 예를들면, 독립변수로 활용할 수 있는 2점슛,3점슛,리바운드는 경기가 진행되어야만 발생한다.
따라서 일반적 머신러닝을 적용하기에는 제약사항이 따르므로 다음과 같은 방안을 고려해 볼 수 있다.

1. 딥러닝 순환신경망, 시계열 LSTM
2. 기간별로 구분된 예측
만약 현재 정규리그 시즌이 끝나고 플레이오프가 진행중이라고 가정해보자.
시즌별 정규리그의 평균값으로 학습데이터의 독립변수로, 각 시즌의 플레이오프 결과(승패차)를 종속변수로 모델을 학습 할 수 있다.
이 모델로 현재까지 진행된 플레이오프 데이터를 예측데이터의 독립변수로 사용할 수 있다.
단 시즌별 정규리그의 평균값의 학습데이터의 수가 KBL역사를 볼 때 적다는 것이 문제이다.

본 분석에서는 예측 데이터없이 가능한 시계열 LSTM을 활용하여 진행한다.

def reorder_teams(df):
    reorder_data = []
    
    # 데이터프레임을 날짜로 그룹화
    grouped = df.groupby('날짜')
    
    for name, group in grouped:
        group = group.reset_index(drop=True)
        matched_indices = set()
        for i in range(len(group)):
            if i in matched_indices:
                continue
            for j in range(i + 1, len(group)):
                if group.loc[i, '팀명'] == group.loc[j, '상대팀명'] and group.loc[i, '상대팀명'] == group.loc[j, '팀명']:
                    reorder_data.append(list(group.iloc[i]))
                    reorder_data.append(list(group.iloc[j]))
                    matched_indices.add(i)
                    matched_indices.add(j)
                    break
    
    # 새로운 데이터프레임 생성
    reorder_df = pd.DataFrame(reorder_data, columns=df.columns)
    
    return reorder_df
