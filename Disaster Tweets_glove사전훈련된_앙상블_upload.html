  1. 대회 과제 (Task)
    목표: 특정 트윗 텍스트를 분석하여, 이 트윗이 실제 재난(화재, 지진, 사고 등) 상황을 묘사하고 있는지(Real Disaster) 아니면 단순히 비유나 일상적인 대화에서 재난 단어를 사용한 것인지(Not Disaster)를 이진 분류(Binary Classification)하는 것이다.
    평가 지표: 모델의 성능은 F1-Score로 평가된다. 이는 정밀도(Precision)와 재현율(Recall)의 조화 평균으로, 클래스 불균형이 있을 수 있는 분류 문제에서 흔히 사용된다.

  2. 데이터 구성 (Dataset Features)
    데이터셋은 CSV 파일 형태로 제공되며, 주요 칼럼은 다음과 같다:
    id: 각 트윗의 고유 식별자
    text: 트윗의 실제 본문 내용
    keyword: 트윗에서 추출된 특정 키워드
    location: 트윗이 발신된 위치 정보. 사용자가 직접 입력하는 값이기에 형식이 일정하지 않고 'dirty'한 데이터가 많다.
    target: (학습 데이터에만 존재) 실제 재난인 경우 1, 아닌 경우 0으로 표시된다.

  3. 모델 전략
    Embedding레이어의 가중치로 사전 훈련된 단어 사전 GloVe를 활용한다.
    두 개의 모델에 적합 후 앙상블하여 최종 예측값을 도출한다.
    모델 1 : Conv1D+LSTM
    모델 2 : Conv1D+GRU

lemmatizer = WordNetLemmatizer()

def clean_tweet(text):
    text = text.lower()
    
    # URL, 멘션 정규화
    text = re.sub(r'http\S+|www\S+', ' url ', text)
    text = re.sub(r'@\w+', ' user ', text)

    # 해시태그 제거 (#fire -> fire)
    text = re.sub(r'#', '', text)

    # 숫자 유지 (5 dead 등 중요)
    text = re.sub(r'[^a-z0-9!? ]', ' ', text)

    words = text.split()
    words = [lemmatizer.lemmatize(w) for w in words]

    return ' '.join(words)

all_text = pd.concat([train['text'], test['text']])
all_text_clean = all_text.apply(clean_tweet)

def build_cnn_bilstm():
    inp = Input(shape=(MAX_LEN,))

    x = Embedding(
        VOCAB_SIZE,
        EMBED_DIM,
        weights=[embedding_matrix],
        trainable=True
    )(inp)

    x = Conv1D(128, 3, activation='relu', padding='same')(x)
    x = MaxPooling1D(2)(x)

    x = Bidirectional(LSTM(128))(x)

    x = Dropout(0.4)(x)
    out = Dense(1, activation='sigmoid')(x)

    model = Model(inp, out)
    model.compile(
        optimizer='adam',
        loss='binary_crossentropy',
        metrics=['accuracy']
    )
    return model

def build_cnn_bigru():
    inp = Input(shape=(MAX_LEN,))

    x = Embedding(
        VOCAB_SIZE,
        EMBED_DIM,
        weights=[embedding_matrix],
        trainable=True
    )(inp)

    x = Conv1D(128, 3, activation='relu', padding='same')(x)
    x = MaxPooling1D(2)(x)

    x = Bidirectional(GRU(128))(x)

    x = Dropout(0.4)(x)
    out = Dense(1, activation='sigmoid')(x)

    model = Model(inp, out)
    model.compile(
        optimizer='adam',
        loss='binary_crossentropy',
        metrics=['accuracy']
    )
    return model
